# JIN-and-KIM 
## <안드로이드 및 웹 기반 달력 프로젝트>

__<최종 목표>__  
- 웹과 안드로이드 html,css,js,java를 기반으로 하여 DB를 연동하는 것을 목표로 함.
- 추후에 가능하다면 서버연결 시도해보기(AWS 등을 사용)  
- 안드로이드 - 웹 연계 달력 만들기
- 데이터베이스는 백석대학교 공지사항 기반으로 하며, 추후 변경 가능케 설정
